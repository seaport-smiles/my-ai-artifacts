How to Build This in Wave AutoPilot
Step-by-step build instructions
Setup ▼
Access AutoPilot from the main Wave screen → Campaign Builder tab → Custom tab → click + to add new campaign.
Step 1 — Create the Campaign
Name: "ISB Clearance Chase" · Trigger: ISB appointment created · Add description · Click Save
Step 2 — Activate the Campaign
Click the Inactive button → changes to Active. Blue dot will appear next to campaign name.
Step 3 — Add Campaign Items
Click + on right side of campaign to add each item. For each: choose Type → fill Details → set When → click Done → check Active box.
Step 4 — Set Timing for Each Item
Day 0 items: When = No Delay
Day 4 text: When = 4 Days After Trigger
Day 7 items: When = 7 Days After Trigger
Day 14 items: When = 14 Days After Trigger
Step 5 — Add Status Change Stop Trigger
Add a Campaign Item → Type: Status Change · New Status: "Needs Rx / Review" · When: triggered by status update. This auto-cancels remaining items when clearance is received.
Step 6 — Add Message Content
For Email items: complete Details section first, then click Email button to enter content before clicking Done. For Text items: paste message into Text Message box. Use merge fields for patient name, dentist name etc.
Step 7 — Activate All Campaign Items
Each item must be individually activated. Check the box under the Active column next to each Campaign Item Name.

Why 3 Weeks Is the Rule — The Real Timeline
Clinical rationale for the minimum clearance window
Reference ▼
The rule: Dental clearance must be received at least 3 weeks before the ISB appointment. This is the minimum time required for Dr. Osborn to plan the case AND for Align to fabricate and ship the aligners. There is no flexibility below this threshold.
Phase 1
Clearance received + scan validity confirmed
Day 0
TC → notifies Dr. Osborn
Phase 2
Rx submitted → ClinCheck returned → review + revisions
~1 week
Dr. Osborn + Align CADCAM
Phase 3
CRS completed → ClinCheck approved
End of week 1
Dr. Osborn
Phase 4
Fabrication + UPS shipping to office
10 business days
Align Technologies
Why no planning happens before clearance: Clearance confirms two things at once — (1) periodontal and restorative health is stable, and (2) tooth anatomy is not changing. If a patient has pending restorative work, the tooth shape can change — invalidating the existing scan. A new scan would be required and any planning done on the old scan cannot transfer. Dr. Osborn would redo all work at no additional compensation. Clearance is the gate.
Revision cycles add time: ~75% of cases require at least one feedback round to Align's CADCAM team. Each cycle adds 1–4 business days. The 1-week planning estimate assumes best-case.

Case Readiness Status Ladder
Wave Ortho Lab Case status — TC keeps this current at all times
Reference ▼
Awaiting Clearance
Contract signed. ISB reserved. Clearance requested but not received. No clinical planning in progress. AutoPilot campaign running.
Clearance Received
Dental clearance in hand. Scan confirmed valid. TC notified Dr. Osborn via Wave Task. AutoPilot campaign still running until TC updates to next status.
Needs Rx / Review
TC updated status → AutoPilot campaign auto-cancels. Dr. Osborn flagged. Rx to be submitted or ClinCheck in progress.
Planning Complete
CRS completed. ClinCheck approved. Case sent to fabrication. 10 BD clock starts.
Ready to Start
Aligners received in office. Logged in Wave. ISB appointment confirmed.

The Start Reservation Concept
Not yet implemented in Wave — documented for future decision
Concept ▼

The root cause of ISB chaos: scheduling a confirmed appointment before clinical readiness. The fix is separating reservation from confirmation. Patient leaves with a date and feels committed. Internally it's a reservation — converts to confirmed ISB only when clearance + ClinCheck + aligners are in office.

Proposed Wave Status
START RESERVED – AWAITING CLEARANCE
Converts to ISB CONFIRMED when: clearance received + ClinCheck approved + aligners in office
